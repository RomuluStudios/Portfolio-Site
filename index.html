Featured Projects

GradePilot

GradePilot is currently in early testing as a desktop app while being converted into a full SaaS web build.


This was a solution I created to solve a very specific problem: students overwhelmed with managing assignment due dates, and the mental fatigue of deciding what to work on and when.


The main features of GradePilot (as of V1.0 testing):

  • Cloud based OCR for coursework importing
  • Adding of class/work schedule
  • Assignment due dates shown clearly on home screen
  • Recommendation engine for prioritizing tasks
  • Current and projected grades

An in depth overview of GradePilot can be found on my Web Dev Projects page .
